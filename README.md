# ScreenShotTogglerApp

Install the dependencies and devDependencies and start the server.

```sh
$ npm install
$ react-native run-android/react-native ios
```
# Test Implementation
I used react-test-renderer, jest  for  implement Testing 

```sh
$ npm run test
```
